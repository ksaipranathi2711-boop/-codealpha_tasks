# Financial Health Dashboard

## Project Overview
This project is developed as part of the CodeAlpha Power BI Internship.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- DAX

## Dataset
Sample Superstore Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Cost KPI
- Profit Margin
- Sales by Category
- Sales by Segment
- Monthly Sales Trend
- Sales by Region
- Interactive Filters

## Business Insights
- Technology category generated the highest sales.
- Consumer segment contributed the highest revenue.
- West region earned the highest profit.
- Discounts negatively impacted profitability.
- Office Supplies showed stable sales performance.
