# 📊 Educational Performance & Resource Allocation Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed using the **Students Performance in Exams** dataset. The dashboard provides insights into student performance based on gender, parental education, lunch type, race/ethnicity, and test preparation course.

The objective is to help educators and institutions analyze academic performance and identify factors affecting student outcomes.

---

## 📂 Dataset

Dataset: Students Performance in Exams

The dataset contains information about:
- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## 📈 Dashboard Features

### KPI Cards
- Total Students
- Average Math Score
- Average Reading Score
- Average Writing Score

### Visualizations
- Average Math Score by Gender
- Average Reading Score by Test Preparation Course
- Average Writing Score by Parental Education
- Student Performance by Lunch Type
- Students by Race/Ethnicity
- Gender Distribution

### Interactive Slicers
- Gender
- Lunch
- Test Preparation Course
- Race/Ethnicity
- Parental Level of Education

## 📊 Key Insights

- Female students perform slightly better in reading and writing.
- Students who completed the test preparation course generally achieve higher scores.
- Students with parents having higher educational qualifications tend to perform better academically.
- Lunch type has a noticeable impact on student performance.
- Student distribution varies across different race/ethnicity groups.
