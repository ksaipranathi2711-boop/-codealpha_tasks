# HR Analytics Dashboard

## 📌 Project Overview

This project is an interactive Human Resources Analytics Dashboard developed using Microsoft Power BI. It provides insights into employee demographics, attrition trends, salary distribution, job roles, and workforce performance to support HR decision-making.

## 📊 Dashboard Features

- Total Employees KPI
- Attrition Count
- Attrition Rate
- Average Monthly Income
- Employees by Department
- Employees by Gender
- Employees by Education Field
- Average Monthly Income by Job Role
- Employees by Years at Company
- Interactive Slicers (Department, Gender, Job Role, Marital Status)

## 🛠️ Tools Used

- Microsoft Power BI
- IBM HR Analytics Employee Attrition & Performance Dataset
- Microsoft Excel / CSV
